# Auth Implementation

## Changes
- **Backend (`src/auth_path.js`):**
    - Added `POST /signup` with password hashing (scrypt).
    - Added `POST /login` with password verification.
    - Added `GET /config` to expose `googleClientId`.
    - Imported `crypto` and `promisify`.
- **Frontend (`dist/login.html`):**
    - Added Google Sign-In script and initialization logic.
    - Updated `handleSignup` to use the new API.
    - Updated `handleLogin` to use the new API.
    - Added logic to fetch `googleClientId` from `/api/auth/config`.
