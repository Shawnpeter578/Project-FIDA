# Change Log - 2026-02-03 00:03

## Event Privilege & Features
*   **File:** `src/event_path.js`
*   **Changes:**
    *   **Create Event (`POST /`):** 
        *   Restricted to `organizer` role.
        *   Added support for `allowArtistApplications` field (boolean).
        *   Initialized `artistApplications` and `confirmedArtists` lists.
    *   **Join Event (`POST /join`):**
        *   Restricted to `user` role.
    *   **Artist Application (`POST /apply`):**
        *   **New Endpoint:** Allows users with `artist` role to apply to events.
        *   Checks if the event allows applications.
        *   Prevents duplicate applications.
        *   Updates the event's `artistApplications` list and the user's `appliedEvents` list.
