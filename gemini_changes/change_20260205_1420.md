# Update ClubApp Mini

## Changes
- **clubapp-mini/dist/script.js**:
    - Updated `BASE_URL` to point to the local backend (`http://localhost:3000/api`).
    - Modified `events.getAll` to accept `page` and `limit` parameters.
    - Implemented real `profile.loadDetails` (alias for `auth.getMe`) and `profile.update`.
- **clubapp-mini/dist/index.html**:
    - Implemented "Load More" pagination logic in `fetchAndRenderFeed` and `renderFeed`.
    - Added a "LOAD MORE" button that conditionally appears when more events are available.
    - Updated `saveProfile` to call `FidaAPI.profile.update`.
    - Refactored `loadProfileData` to map fields correctly from the backend response.

## Rationale
To ensure the "mini" version of the application remains functional and consistent with the recent backend upgrades (pagination, profile management).
